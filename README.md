## Deployment

to install all dependecy,

```bash
 npm install 
```
or

```bash
 yarn install 
```


To start this project locally

```bash
  npm run start
```
or
```bash
  yarn start
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`SERVER_PORT`

`PG_PORT`

`DB`

`DB_USERNAME`

`DB_PASSWORD`

`SERVER_PORT`

`ROOT_PASSWORD`

`NODE_ENV`

`JWT_SECRET_KEY`

`JWT_EXPIRE_DURATION`
